# 🎬 Sakila DVD Rental Analytics

## 📖 Overview

This project analyzes the Sakila DVD Rental dataset to extract meaningful business insights using SQL and Power BI. The objective is to understand customer behavior, rental trends, sales performance, and inventory distribution through exploratory data analysis and interactive dashboards.

## 🛠️ Tools Used

* SQL (MySQL Workbench)
* Power BI
* Excel

## 📊 Key Analysis

* Customer segmentation and behavior analysis
* Sales and revenue trend analysis
* Rental activity and usage patterns
* Inventory distribution across stores and categories
* Film category, rating, and pricing analysis

## 📈 Dashboard Features

* Interactive Power BI dashboards with multiple pages
* KPI cards (Total Revenue, Customers, Rentals, Films)
* Monthly rental and revenue trend analysis
* Customer distribution by city and country
* Inventory and film performance visuals
* Navigation buttons for seamless user experience

## 💡 Key Insights

* A large portion of customers are repeat users, indicating strong customer retention
* Revenue and rental activity vary across locations, highlighting key markets
* Inventory is evenly distributed across stores but concentrated in specific categories
* Most films are English-language, showing limited language diversity
* Certain film categories and ratings dominate rental demand

## 🚀 Conclusion

This project demonstrates how data-driven insights can improve decision-making in a rental business. By analyzing customer behavior, optimizing inventory distribution, and identifying revenue trends, businesses can enhance operational efficiency and customer satisfaction.
